Credit Card Fraud Detection System

Credit Card Fraud Detection System

Welcome to our Credit Card Fraud Detection System. This project leverages advanced technologies to enhance financial security by accurately detecting fraudulent transactions.

Project Overview

This project consists of the following key components:

  • Data Collection and Preprocessing
  • Model Building and Evaluation
  • Model Deployment
  • Geolocation Identification
  • User and Bank Notifications
  • User Feedback Loop
  • Testing the Code

Data Collection and Preprocessing

We use a dataset of credit card transactions, performing initial exploration, data balancing using SMOTE, and feature scaling with StandardScaler.

Model Building and Evaluation

We train Logistic Regression and Random Forest models, evaluate their performance using various metrics, and select the best-performing model.

Model Deployment

We save the best model using joblib and deploy it on AWS for scalability and accessibility, integrating it with user interfaces.

Geolocation Identification

We capture geolocation data and compare the transaction location with the user's current location to validate transactions and detect potential fraud.

User and Bank Notifications

We notify users of successful and potentially fraudulent transactions via email/SMS, implementing multi-factor authentication (MFA) and using behavioral biometrics for added security.

User Feedback Loop

We provide a feedback mechanism for users to confirm or deny transactions, helping to fine-tune the fraud detection system.
